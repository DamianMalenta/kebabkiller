# 📖 SŁOWNICZEK — „komputerowe" słowa po ludzku

> Nie musisz tego umieć na pamięć. Wracaj tu, gdy spotkasz nieznane słowo.

| Słowo | Po ludzku |
|-------|-----------|
| **Repozytorium / repo** | Folder z projektem, którego historię pamięta Git. Jak segregator z wersjami. |
| **Git** | Program, który zapisuje historię zmian i pozwala je cofać. „Wehikuł czasu" dla kodu. |
| **Gałąź (branch)** | Osobna ścieżka pracy, żeby próbować zmian bez ruszania wersji głównej. Jak brudnopis. |
| **Pull** | Pobranie najnowszej wersji z internetu na swój komputer. |
| **Commit** | Zapisany „punkt w czasie" — zestaw zmian z opisem. |
| **PR (Pull Request)** | Propozycja zmian do zatwierdzenia, zanim trafią do wersji głównej. Jak „prośba o akceptację". |
| **Rollback** | Cofnięcie zmiany do stanu sprzed. Bezpiecznik. |
| **Agent AI** | Twój pomocnik, który czyta kod, planuje i (po Twoim OK) pisze/poprawia. |
| **Audyt** | Dokładne zbadanie kodu: co działa, co jest słabe, co wyrzucić. |
| **MVP** | Najmniejsza działająca wersja produktu — pierwszy realny kawałek. |
| **Produkcja / prod** | Wersja programu, która naprawdę działa „na żywo" dla użytkowników. |
| **Self-hosting / samohosting** | Trzymanie programu na własnym serwerze/komputerze, nie u obcej firmy. |
| **LLM / model** | „Mózg" AI, który rozumie i pisze tekst/kod (np. darmowe modele z Groq, Gemini). |
| **Darmowy tier** | Darmowy limit usługi (np. tyle zapytań dziennie za 0 zł). |
| **MCP** | Standard, którym agent „podłącza się" do programu, żeby widzieć jego pliki/dane. |
| **Moduł / adapter** | Mały dodatek wpinany do programu, który daje mu nową funkcję (tu: agenta). |
| **Tożsamość (w Symbioncie)** | Zestaw informacji o danym programie, dzięki którym agent „zna" jego zasady i granice. |
| **ADR** | Krótka notatka: „podjęliśmy taką decyzję, bo...". Żeby nie zapomnieć dlaczego. |
| **Symbiont** | Nazwa robocza naszego produktu docelowego — agenta wpinanego w rdzeń programów. |
| **gema0** | Twój starszy prototyp — materiał wejściowy, który audytujemy. |
| **macius** | Twoje centrum dowodzenia — folder, w którym dzieje się cała nowa praca. |
